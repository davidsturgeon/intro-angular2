Some Thoughts

Still very experimental, things changing a lot

Router not fully ported/documented yet

Forms module has not matured

Preparing for Angular 2

  • Use ES6!
  • Make everything a directive (Component Approach)
  • Use controllerAs syntax
  • Avoid $scope.$watch, use reactive approaches
  • Only create services (vs. factories)
  • Use ES6 classes for controllers and services
  • Use the new router for 1.4
